# 2018

## 为什么明明知道是正确的事情, 却做不到

翻看2018的记录, 一条清晰的脉络是 「为什么明明知道是正确的事情, 却做不到」, 2018全年在各个方面都有纠结这个问题, 现在回顾来看, 几乎是**完败**, 所以 2019 还是需要在 **事上练**, 所幸因为这一年的经历, 信心更足了一些.

## 精简与反馈->执行

2018经历的另一件事是生活理念上的改变, 崇尚精简与反馈, 现在回顾来看, 整个历程可以概括为: 清空 -> 度过少所带来的空虚 -> 优化执行 -> 更加充实. 举几个例子:

- 住: 找到 **最小可用**, 然后执行, 之后保持清洁和保持条理都非常容易
- 看书: 家里书籍读完, 设置保留书籍的上限, 全面电子化(纸质书处理, wiki中书籍记录转到豆瓣和blog)
- 技术: 主线是构建技术栈/技术技能树, 基于积累的wiki, 并把wiki打造成对自己友好的产品, 行动上也是对wiki进行大规模的删减, 并给知识配置上索引

## 平衡自己

2018一个很重要的事情, 其实是平衡自己 -- 经历过连续3年的高速增长, 怎么面对 2018 的增长放缓? 好像和主旋律连续起来了, 但个人和集体, 和社会, 就是会用这样的共鸣. 上面的经历, 也是在各个方面尝试磨炼自己, 一方面要理解, 思考清楚里面的本质, 不至于在增速的表象里面继续迷失与纠结; 一方面要做相应的改变, 高增速下存在粗放, 有一些东西被掩盖掉了, 整个理解的过程也是在问询自己, 改变自己
